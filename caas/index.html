Constellation-as-a-Service

Case 1: Simple Strategy Comparison

Below, we demonstrate a simple case where focusing on maximizing global throughput generates unfair and different assignments compared to the max-min scheduling algorithm.
Each satellite in this constellation carries 200 units of the computing resource that both users ask for.
The green user demand requests 1000 units of compute (e.g., CPU or GPU), and the orange user demand requests 300 units of the same resource.

Maximize Throughput

The constellation maximizes global throughput.

  • User Green: Demanded 1000, assigned 1000
  • User Orange: Demanded 300, assigned 200

Max-Min Fairness

The constellation uses a max-min fairness strategy.

  • User Green: Demanded 1000, assigned 800
  • User Orange: Demanded 300, assigned 300

Case 2: Strategy Comparison at 2 Locations

Here, we present the case where users in 2 locations competing for resources, scaling up from the previous simple case.

  • Each satellite is equipped with 256 units of memory and 200 units of computing power.
  • For the case in Asia, users have the same demands as the previous case, where the blue user demands 1600 units of computing power and 1024 units of memory, while the green user demands 100 units of memory.
  • The second case is on the U.S. East Coast, where the green user requested 700 units of memory, while the red user only requested 20 units of memory.

Maximize Throughput

Maximize global throughput

  • User Blue (Asia): Demanded 1600 units of compute, assigned 1200; demanded 1024 units of memory, assigned 1536
  • User Green (Asia): Demanded 100 units of memory, assigned 0
    User Green (U.S. East Coast): Demanded 700 units of memory, assigned 600.
  • User Red (U.S. East Coast): Demanded 20 units of memory, assigned 0

Max-Min Fairness

Max-min fairness strategy

  • User Blue (Asia): Demanded 1600 units of compute, assigned 1000; demanded 1024 units of memory, assigned 1280
  • User Green (Asia): Demanded 100 units of memory, assigned 200
    User Green (U.S. East Coast): Demanded 700 units of memory, assigned 400.
  • User Red (U.S. East Coast): Demanded 20 units of memory, assigned 200
